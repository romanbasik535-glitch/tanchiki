# Zombie Mobile Shooter (Godot 4)

Базовая структура проекта 3D шутера про зомби для телефона (альбомная ориентация).

## Что уже заложено
- Главное меню (`Старт`, `Настройки`, `Выход`)
- Экран настроек (качество графики)
- Игровая сцена с загрузкой карт
- 3 карты: `Map01`, `Map02`, `Map03`
- Игрок с заготовкой под 5 видов оружия:
  - Pistol
  - Shotgun
  - SMG
  - Rifle
  - Sniper
- Подбор аптечки
- Подбор оружия
- Базовый враг-зомби
- Глобальное состояние через `autoload/GameState.gd`

## Структура
- `project.godot` — настройки проекта и mobile renderer
- `autoload/GameState.gd` — состояние текущего забега
- `scenes/main/Main.tscn` — главное меню
- `scenes/ui/SettingsMenu.tscn` — настройки
- `scenes/game/Game.tscn` — игровая сцена
- `scenes/maps/` — 3 карты
- `scenes/player/Player.tscn` — игрок
- `scenes/enemies/Zombie.tscn` — зомби
- `scenes/items/` — аптечка и оружие
- `scripts/` — логика по модулям
